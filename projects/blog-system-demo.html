Szybki start

# Klonuj repository
git clone https://github.com/danielkmita94-jpg/blog-system.git
cd blog-system
# Zainstaluj zależności
composer install
# Skopiuj .env.example do .env i skonfiguruj
cp env.example .env
# Uruchom w XAMPP
# Otwórz: http://localhost/blog-system/public/
