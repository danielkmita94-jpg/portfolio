## 🚀 **JAK URUCHOMIĆ DEMO:**

### **Wymagania:**
- XAMPP (Apache + MySQL)
- PHP 7.4+
- MySQL 8.0+

### **Kroki:**
1. **Uruchom XAMPP** - Apache i MySQL
2. **Skopiuj projekt** do `htdocs/InvoiceSystem/`
3. **Utwórz bazę** `invoice_system`
4. **Skonfiguruj** `.env` z danymi bazy
5. **Otwórz** `http://localhost/InvoiceSystem/public/`
